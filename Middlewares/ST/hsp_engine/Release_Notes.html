Purpose

The HSP Engine Middleware allows to configure and process complex signal processing.

It provides some modules:

  • DSP Library: Set of DSP functions,
  • CNN Library: Set of CNN functions,
  • Direct Library: Set of functions to execute Direct Command provided by the HSP Firmware,
  • ProcList Library: Set of functions used to record a processing list,
  • Firmware: Set of HSP Firmwares & Plugins
